# D'Cade

> A personal system-level presence component for Windows & macOS.

D'Cade is not an app you "open" — it's always there, quietly present, instantly accessible when you need it.

## System Requirements

- **Windows**: Windows 10/11 (64-bit)
- **macOS**: macOS 12+ (Intel & Apple Silicon) — coming soon

## Installation

### Windows

1. Download the `.exe` installer
2. Run the installer (SmartScreen may show a warning — click "More info" → "Run anyway")
3. Follow the installation wizard
4. D'Cade will start automatically and appear in your system tray

### First Launch

- **Orb**: A small floating orb appears — this is your entry point
- **Hotkey**: Press `Ctrl + `` ` (backtick) to show/hide D'Cade
- **Tray**: Right-click the tray icon for options

## Features

- **Always Present** — Starts with your system, stays out of your way
- **Instant Access** — One hotkey to capture thoughts
- **Note-first Design** — Just write, no folders or tags to manage
- **Local-first** — Your data stays on your device, encrypted
- **Cloud Sync** (Pro) — Sync across devices with E2E encryption
- **AI Commands** (Pro) — Voice-to-text, time navigation, context restore
